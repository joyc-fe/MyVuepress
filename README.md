# taopoppy的学习博客

这里包含了前端的各种学习资料和知识讲解

博客以挂载到github,网站为[www.taopoppy.cn](taopoppy.cn)，但是由于github网站很慢，后续我会考虑挂载到国内的网站

你可以直接选择克隆到本地：
```javascript
git clone https://github.com/taopoppy/MyVuepress.git

cd MyVuepress
npm run docs:dev
```
然后就能在浏览器的8080端口访问
